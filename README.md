# About

DrevaEggs.com is a simple website built to allow the players on the Dreva server of Project: Gorgon to know when the fuck the egg run is. Since the server uses us eastern time, times are given in that. you can adjust your offset to match.

# Features

- Current server time
- Scheduled eggruns for the server
- Countdown until next eggrun
- User-configrable time offset.
- Egg door open/closed status monitoring
- Portal code reminder
- Egg run rules of the road
